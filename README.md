Aroumarket-e-commerce est une application e-commerce que j'ai dévéloppé avec Laravel.

## Installation
```sh
composer install
```
### Compilation et rechargement automatique en mode développement

```sh
npm run dev
```

### Compiler et minifier pour la production

```sh
npm run build
```
