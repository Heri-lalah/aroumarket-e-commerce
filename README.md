# Aroumarket
Aroumarket-e-commerce est une application e-commerce que j'ai dévéloppé avec Laravel.

## Installation

```bash
git clone https://github.com/Herilala-rakotonasolo/aroumarket-e-commerce.git
```

```bash
composer install
```

```bash
npm install
```

```bash
php artisan migrate --seed
```
